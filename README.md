## Overview

This repository is a curated collection of projects built using **Amazon Web Services (AWS)**. It serves as a cloud portfolio showcasing hands-on experience with AWS services, cloud infrastructure, deployment strategies, security, networking, and database management.

The repository is continuously updated as I explore new AWS services and build practical, production-oriented cloud applications.

---

## Objectives

* Gain practical experience with AWS Cloud services.
* Build and deploy real-world cloud applications.
* Learn cloud architecture and infrastructure management.
* Implement secure and scalable solutions.
* Document cloud deployment workflows and best practices.

---

## AWS Services

This repository includes projects and experiments involving services such as:

| AWS Service           | Purpose                                     |
| --------------------- | ------------------------------------------- |
| Amazon EC2            | Virtual servers and application hosting     |
| Amazon RDS            | Managed relational databases                |
| Amazon S3             | Object storage and static website hosting   |
| AWS IAM               | Identity and Access Management              |
| Amazon VPC            | Networking and resource isolation           |
| Security Groups       | Network security and firewall configuration |
| AWS CloudShell        | Browser-based command-line environment      |
| AWS CloudWatch        | Monitoring and logging                      |
| Elastic Load Balancer | Traffic distribution                        |
| Auto Scaling          | Automatic resource scaling                  |
| AWS Lambda            | Serverless computing                        |
| API Gateway           | API management                              |
| CloudFront            | Content delivery network                    |
